# OIBSIP - Online Examination System (Console)

A console-based Online Examination System developed for AICTE OIBSIP Internship demo.
Features:
- Login
- Update profile and password
- Selecting answers for MCQs
- Timer and auto-submit (demo 60 seconds)
- Logout / Close session

## How to run
1. Compile:
```
javac User.java Question.java OnlineExam.java
```
2. Run:
```
java OnlineExam
```
